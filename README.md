## 2. 為何要使用Docker?(Why Docker?)

- **建立虛擬環境**
- **建立完全獨立環境專案,並提供該專案所有附屬套件**
- **可建立多個服務,如資料庫服務**
- **容易部署和發佈**
- 讓共同開發者在不同平台有**相同開發環境**
